# START HERE

For a normal user:

1. Read `GITHUB_DOWNLOAD_INSTRUCTIONS.html`.
2. Create a virtual environment.
3. Install `requirements.txt`.
4. Run `python run.py`.
5. Open `http://127.0.0.1:5007`.

For GitHub publishing:

- Upload this whole folder into the repository so the README is visible.
- Attach `SoulAgentOS-v0.13.zip` as a Release asset.
- Tell users to download from **Releases → Assets**, not to clone.
